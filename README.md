# TinDog Project

## About the Project

TinDog is a Udemy practice project for building a responsive landing page using Bootstrap and custom CSS.

The webpage is for a fictional dog dating app. It includes a title section, feature section, testimonial section, press logos, pricing cards, and a footer.

## Live Website

View the published webpage here:

[Live Demo](https://dorraboussetta.github.io/tindog-project/)


## Technologies Used

- HTML
- CSS
- Bootstrap

## What I Learned

- Building a full landing page layout
- Using Bootstrap containers, rows, and columns
- Using Bootstrap buttons, cards, and utility classes
- Creating responsive sections with Bootstrap Grid
- Adding SVG icons inside buttons and feature blocks
- Creating a gradient background with custom CSS
- Linking an external CSS file
- Making images responsive with Bootstrap classes
- Structuring a webpage into clear sections
- Practicing responsive design for desktop and mobile screens
